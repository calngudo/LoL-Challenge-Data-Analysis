# League of Legends Champion Win Challenge – Data Insights Project
# Overview
- This project analyzes match data for a group of friends attempting a community challenge:
winning at least one ranked Flex game with every champion without
- Using the Riot Games API, the project retrieves, cleans, and processes match data to generate insights about each player’s progress and playstyle. The goal is to build a structured, insightful dataset while gaining experience with:
- Riot API usage
- Data retrieval and cleaning pipelines
- Python data analysis
- Using Docker & Flask for deployment
- Azure for hosting / automation
- Microsoft BI
